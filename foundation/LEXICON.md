# LifeBook Lexicon

> "Words shape culture. Culture shapes products."

| Instead of... | We say... | Why |
|--------------|-----------|-----|
| User | Storykeeper | People are not users. They are the authors of their own lives. |
| Customer | Storykeeper | The relationship is personal, not transactional. |
| Session | Conversation | People don't have sessions. They have conversations. |
| Chat | Conversation | The interaction is relational, not technical. |
| Memory Record | Memory | Keep language natural. |
| Journal Entry | Moment | Life is experienced in moments. |
| Dashboard | Home (or avoid entirely) | LifeBook is a place, not a control panel. |
| AI Assistant | Companion | Assistance is a function. Companionship is a relationship. |
| Database | Library (when appropriate) | Human language before technical language. |
| Upload | Add | Simpler and more natural. |
| Search | Remember | Reflects LifeBook's purpose. |
| Timeline | Life Journey | More human, less mechanical. |
| Feature | Capability | Focus on what the Storykeeper gains. |
| Notification | Invitation | Respectful, never demanding. |

---

## Vocabulary Principles

- Prefer human language over technical language.
- Prefer verbs that suggest relationship.
- Avoid corporate terminology.
- Avoid software jargon in Storykeeper-facing experiences.
- If a word makes LifeBook sound like an application rather than a companion, choose another.
